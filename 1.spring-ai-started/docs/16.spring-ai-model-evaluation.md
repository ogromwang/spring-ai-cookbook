# 模型评估

模型评估（Model Evaluation）用于评估 AI 模型的输出质量，帮助优化提示词和模型选择。

::: code-group

```java [java:基本用法]
@Autowired
private EvaluationModel evaluationModel;

// 评估模型输出
EvaluationResponse response = evaluationModel.evaluate(
    "原始问题",
    "模型回答",
    "期望答案"
);

double score = response.getScore();
```

:::
